# LAMP Docker Development Environment
 
 Simple environment for developing LAMP based stuff.

 Set the environment variable `PROJECT_PATH`

 `PROJECT_PATH=YOUR_PROJECT_PATH docker-compose up`

 
| Service       | Port          |
| ------------- | ------------- | 
| Apache        | 8080          | 
| PhpMyAdmin    | 8081          | 
| MariaDB       | 3306          |
